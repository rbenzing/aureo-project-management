Quick Start Guide

Get SlimBooks running locally in minutes

Prerequisites

Required Software

  • Node.js 16+ and npm
  • PostgreSQL 12+
  • Git

Optional Tools

  • Docker & Docker Compose
  • VS Code (recommended)

1 Clone the Repository 

git clone https://github.com/rbenzing/slimbooks-app.git
cd slimbooks-app

2 Set Up Environment Variables 

cp .env.example .env
# Edit .env with your database credentials

3 Install Dependencies 

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

4 Database Setup 

# Create database
createdb slimbooks

# Run migrations
cd backend
npm run db:migrate

# Seed sample data (optional)
npm run db:seed

5 Start Development Servers 

# Terminal 1: Start backend
cd backend
npm run dev

# Terminal 2: Start frontend
cd frontend
npm run dev

The app will be available at http://localhost:3000

Docker Quick Start

Prefer Docker? Get up and running with a single command:

docker-compose up -d
